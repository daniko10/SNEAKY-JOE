# SNEAKY-JOE
Sneaky-Joe is a singleplayer game where user becomes a 'Joe' and starts collecting vodka as points. Main goal is to achieve as many vodkas as possible. As a player 
you are obliged to avoid rockets which are being randomly generated at comes towards you! Items are spawned (extra hearts include) on levels which you have to get through.
